Bootstrap Accordion

This is the first accordion panel's content. It is shown by default, as the .show class is added. You can place any content you like within this .accordion-body, including text, images, lists, and more.

This is the second accordion panel's content. Notice the .collapsed class on the button and the absence of the .show class on the .accordion-collapse. This means it will be initially hidden. Click the header to toggle it.

And here's the third accordion panel! Just like the second one, it starts collapsed. You can add as many accordion items as you need to organize your content effectively. Remember to keep the data-bs-parent attribute consistent with the main .accordion container's ID.
